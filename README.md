# vim-marktext

Vim ❤️ marktext

forked from [GitHub - wookayin/vim-typora](https://github.com/wookayin/vim-typora)

## Usage

```
:Marktext
```

to open the current file in Mark Text (macOS only).

Vim buffers and MarkText windows will synchronize upon every save.
